# TypeScript Reference Kit

This is a runnable reference adapter for the Inner Core contracts. It is intentionally dependency-free and does not require Fastify, Vite, React, SQLite, or any game engine.

It demonstrates:

- health payload creation
- release catalog creation
- Prometheus metric rendering
- maintenance bypass and response rules
- sanitized environment loading
- product analytics envelope and privacy guard
- persistence ownership planning
- signed co-op/session tickets
- visual timeline event merging and elapsed-aware animation delays
- a minimal route-registration adapter shape

Run from the repo root:

```bash
npm run test
```

These files can be copied into a game repo and adapted to that game's framework.
